Eye Icon Password Data Protector

Hide and Display fields easily using eye icons by clicking on it

By default odoo hides the password but this widget provide the way to display the password

Or else you can hide this field
In just one click

Features

  • 👁️‍🗨️ Hide Password by Default

    Automatically masks password fields on Login, Signup, and Reset Password pages.

  • 🛠️ Custom Field Compatibility

    Easily apply to any custom password field using: < field name="your_field_name" widget="password_hide"/>

  • 🔄 Toggle Visibility

    Includes a toggle icon to show or hide password input dynamically.

  • ⚙️ Lightweight & Reusable

    Built as a standalone widget, ready to be reused across modules.

  • 🧩 Easy Integration

    No extra configuration required—just plug and play with the widget attribute.

  • 🎯 Secure by Design

    Ensures sensitive inputs are hidden by default to enhance user security.

  • 🎨 UI-Friendly Design

    Smooth and intuitive interface, matches standard Odoo styling.
